정적 앱 만들기 난이도 쉬움

뽑기 룰렛, 30분이면 만듭니다

HTML 파일 하나로 되는 가장 쉬운 앱이에요. AI에게 시키고, 받아서, 올리기. 오늘은 ‘학생 뽑기 룰렛’을 처음부터 배포까지 함께 만들어봅니다. 아래에서 완성 모습을 먼저 돌려보세요.

완성 예시 · 직접 돌려보세요
이름을 넣고 돌려보세요

이걸 6단계로 만들어봅니다

코드를 직접 짜지 않아요. 무엇을 만들지 정하고, AI에게 정확히 요청하고, 받아서 올리기만 하면 됩니다. 순서대로 따라오세요.

무엇을 만들지 한 줄로 정합니다

거창할 필요 없어요. 이렇게 한 문장이면 충분합니다.

“이름을 여러 개 넣고 버튼을 누르면 한 명을 무작위로 뽑아주는 룰렛.”

데이터를 저장할 필요가 없죠? 그래서 정적 앱, HTML 파일 하나로 끝납니다.

AI에게 이렇게 요청합니다

아래 문장을 그대로 복사해서 Claude(또는 쓰시는 생성형 AI)에 붙여넣으세요. 구체적으로 적을수록 결과가 정확해집니다.

AI에게 보낼 요청 
HTML 파일 하나로 동작하는 '학생 뽑기 룰렛'을 만들어줘.

- 텍스트 상자에 이름을 여러 줄로 입력할 수 있어
- '돌리기' 버튼을 누르면 이름들이 빠르게 바뀌다가 점점 느려지며 한 명에서 멈춰
- 뽑힌 이름은 크고 눈에 띄는 색으로 보여줘
- 휴대폰에서도 잘 보이게 반응형으로 만들어줘
- 전체를 index.html 하나에 담아서, 복사해서 바로 쓸 수 있는 완성된 코드로 줘
팁 — 원하는 게 더 있으면 줄을 추가하면 돼요. “배경은 파란 계열로”, “결과가 나오면 종이 조각이 떨어지는 효과”처럼요.

받은 코드를 파일로 저장합니다

AI가 준 코드를 통째로 복사해, 코드 에디터(VS Code)나 메모장에 붙여넣고 index.html이라는 이름으로 저장하세요.

이때 파일 이름 끝이 반드시 .html이어야 합니다. 메모장으로 저장할 땐 ‘파일 형식’을 모든 파일로 바꾸고 index.html로 저장하면 돼요.

열어서 확인합니다

저장한 index.html 파일을 더블클릭하면 브라우저에서 바로 열립니다. 이름을 넣고 ‘돌리기’를 눌러보세요.

VS Code를 쓴다면 Live Server 확장으로 열면, 코드를 고칠 때마다 화면이 자동으로 새로고침돼서 편해요.

마음에 안 들면 고칩니다

완벽하지 않아도 괜찮아요. 바꾸고 싶은 걸 말로 다시 요청하면 됩니다. 이런 식으로요.

“이름이 뽑힐 때 더 크게 확대되면서 색이 바뀌게 해줘”
“제목을 ‘오늘의 발표자’로 바꿔줘”
“한 번 뽑힌 사람은 다음에 안 나오게 해줘”
팁 — 한 번에 하나씩 바꾸세요. 고치고 → 확인하고 → 또 하나. 그래야 뭐가 바뀌었는지 알 수 있어요.

인터넷에 올립니다 (배포)

index.html이 든 폴더를 통째로 Netlify Drop에 끌어다 놓으면 링크가 생깁니다. 그 링크를 학생들에게 공유하면 끝이에요.

가장 빠른 배포 — Netlify Drop

app.netlify.com/drop 접속 → index.html이 든 폴더를 끌어다 놓기 → ~.netlify.app 주소 완성. 자세한 방법과 다른 선택지(GitHub Pages·Vercel)는 메인의 ‘04 배포하기’를 참고하세요.
